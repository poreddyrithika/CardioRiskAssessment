# CardioRiskAssessment

A **Streamlit-based web application** that predicts the **risk of heart disease** using a Machine Learning model.



## 🌐 Live Demo

👉 **Try the app here:**
🔗 https://cardioriskassessment-hqh7w7kbn82uowmbah4qzr.streamlit.app/




## 📌 Overview

CardioRiskAssessment is an interactive web app that allows users to input medical parameters and instantly get a prediction about their heart disease risk.

It demonstrates how **Machine Learning + Web Apps** can be combined to create real-world healthcare solutions.



## 🚀 Features

* 🧠 Machine Learning-based prediction
* ⚡ Instant results
* 🎯 User-friendly interface
* 🌐 Accessible from anywhere (no installation required)
* 📊 Built with real-world medical parameters



## 🛠️ Tech Stack

* **Frontend/UI**: Streamlit
* **Backend**: Python
* **Machine Learning**: Scikit-learn
* **Libraries**: Pandas, NumPy



## 📂 Project Structure

```
CardioRiskAssessment/
│── app.py                # Main Streamlit application
│── heart_model.pkl       # Trained ML model
│── requirements.txt      # Dependencies
│── README.md             # Project documentation
```



## ⚙️ How It Works

1. User enters health-related inputs
2. The app loads the trained model (`heart_model.pkl`)
3. Model processes the inputs
4. Prediction is displayed instantly



## 💻 Run Locally

Clone the repository:

```
git clone https://github.com/poreddyrithika/CardioRiskAssessment.git
cd CardioRiskAssessment
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
streamlit run app.py
```
